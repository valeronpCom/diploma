Данный документ содержит краткое описание программной части дипломного проекта:
"Веб-приложение по продаже музыкального оборудования".

Целью данного проекта является реализация веб-приложения интернет-магазина по продаже музыкального оборудования.
При создании данного проекта использовались технологии:
- язык программирования Ruby
- фремворк Rails
- JavaScript
- Jquery

При реализации веб-приложения используется реляционная база данных PostgreSQL.

Для развертывания данного приложения необходимо:
- запустить исполняемый файл Shop.exe, который разархивирует полный проект
- перенести содержимое распакованного архива с операционной системы Windows на любую UNIX-систему (например macOS)
Аргументированность данного шага приведена в руководстве пользователя. Также следует добавить, что при попытке
разворачивания приложения на Windows будет невозможно установить некоторые гемы (библиотеки) Ruby, что приведет к
невозможности полноценного функционирования веб-приложения.
- после переноса архива для разворачивания веб-приложения необходимо пройти шаги, указанные в руководстве пользователя.
Также для прохождения инструкции, указанной в руководстве пользователя, необходимо стабильное подключение к сети Интернет.
